## 🎯 **Features**

### ✨ Core Features
- ✅ **Add Tasks** - Quick add with Enter key support
- ✅ **Complete Tasks** - Mark tasks as done with one click
- ✅ **Delete Tasks** - Remove individual tasks
- ✅ **Delete All** - Clear all tasks with confirmation
- ✅ **Auto-Save** - Tasks saved automatically to local storage
- ✅ **Task Counter** - Real-time task statistics

### 🎨 Visual Features
- 🌓 **Dark/Light Mode** - Eye-friendly themes
- 📊 **Progress Bar** - Visual task completion tracking
- 📈 **Live Statistics** - Total, completed, pending counts
- 🎭 **Gradient Design** - Modern gradient backgrounds
- 💫 **Smooth Animations** - Beautiful transitions
- 📱 **Responsive** - Works on all screen sizes

### 🚀 Advanced Features
- 💾 **Local Storage** - Data persists after closing
- 🔍 **Search Tasks** - Find tasks quickly (coming soon)
- 📅 **Due Dates** - Set deadlines (coming soon)
- 🔔 **Notifications** - Reminders for pending tasks (coming soon)
- ☁️ **Cloud Sync** - Sync across devices (coming soon)
